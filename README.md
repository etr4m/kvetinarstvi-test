# Květinářství - Test Decap CMS

Test projektu s Decap CMS pro květinářství.

## Struktura:

- `index.html` - Hlavní stránka
- `admin/` - Decap CMS admin panel
- `_products/` - Markdown soubory s produkty
- `_posts/` - Blog příspěvky (novinky)
